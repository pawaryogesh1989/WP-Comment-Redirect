Use this plugin to redirect to custom page after commenting.

== Description ==

This plugin can be used to redirect to custom page after commenting.

= Details =

* Redirect After Comment To Custom Page.
* Custom Page URL can be set from the plugin settings.
* URL can be page, post or external url.

== Installation ==

The Plugin can be installed in two ways.

= Download and Install =

1. Download and upload the plugin files to the `/wp-content/plugins/wp-comment-redirect` directory from the Wordpress plugin repository.
2. Activate the plugin through the 'Plugins' screen in WordPress admin.

= Using Plugin Interface =

1. Open the plugin Interface in the wordpress admin and click on "Add new".
2. In the Search box enter "Redirect After Comment To Custom Page" and hit Enter.
3. Click on "Install" to install the plugin.
